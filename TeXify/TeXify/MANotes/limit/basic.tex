\subsection{Cheat Sheet}

\ti{常见的等价无穷小. }

(1) 当$x\to 0$的时候, 
第一组:
$$
x\sim \sin x \sim \tan x \sim \arcsin x \sim \arctan x\sim e^x-1\sim \ln(1+x)
$$
第二组(Taylor展开到第二项):
$$
1-\cos x \sim {1\over 2} x^2
$$
第三组:
$$
\sqrt[n]{1+x}-1\sim {1\over n } x
$$
第四组: 
$$
a^x-1 \sim \ln a
$$

\ti{常见三角恒等式. } 
$$
1+\tan^2(x) = \sec^2(x) \qquad 1+\cot^2(x) = \csc^2(x)
$$
